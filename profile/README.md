# XanaScript

Program in Portuguese. Run on WebAssembly. Real performance.

XanaScript is a programming language with Portuguese syntax, optimizing compiler (JavaScript + WebAssembly), built-in ORM, compile-time macros, LSP, and zero runtime dependencies.

## Repositories

| Repository | Description |
|---|---|
| [xs](https://github.com/xanascr/xs) | Core language compiler and CLI |
| [xs-vscode](https://github.com/xanascr/xs-vscode) | VS Code extension |
| [xs-site](https://github.com/xanascr/xs-site) | Official website |
| [xs-examples](https://github.com/xanascr/xs-examples) | Code examples |
| [xs-installer](https://github.com/xanascr/xs-installer) | Windows installer |

## Quick Start

```bash
npm install -g xanascript
xs run app.xs
```
